인트로 스토리

여섯 개의 아름다운 지형으로 구성된 코딩 아일랜드가 바이러스에 감염되어 그곳에 살고 있는 로봇들이 망가지고 섬이 모두 분리되었다.
때마침 세계를 탐험하던 탐험가 제피가 로봇들의 도움 요청 소리를 듣고 바이러스를 퇴치하러 코딩 아일랜드를 향해 날아간다.

메인

1단계 시작 시 분리된 섬.

전체 학습 완료 시 결합된 섬.

핵심개념맵

각 단계를 구성하는 전집 도서를 로드맵으로 구성하여 단계 별 12~14개의 학습이 이루어짐.
캐릭터가 서 있는 발판에 학습 완료, 학습 예정, 학습 중 상태가 표시되며 완료 후 해당 단계의 아이템을 획득.

1단계 요정의 숲

2단계 숨겨진 계곡

3단계 신비의 연못

4단계 신들의 섬

5단계 빛의 광산

6단계 비밀의 정원

쏙쏙 코딩 동화

동화책으로 독서 중 AR 기능을 통해 다음에 이어지는 코딩 퍼즐과 모디 블록의 학습 도구를 획득.

똑똑 코딩 퍼즐

실물 퍼즐을 배열하여 앱에 인식시켜 코딩 활동 실행.

쓱쓱 모디 교실

가이드 영상을 보고 엔트리 프로그램을 활용하여 모디 블록으로 발명품 만들기.

주인공 캐릭터 ‘제피’

8-9세 남자 어린이 탐험가

단계별 로봇 캐릭터
